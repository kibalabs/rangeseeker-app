# Farcaster Frame Assets

This directory should contain the following images for Farcaster mini app support:

- `image.png` - Main frame preview image (recommended size: 1200x630px)
- `splash.png` - Splash screen shown when launching the mini app (recommended size: 1200x1200px)
- `icon.png` - App icon (recommended size: 512x512px)
- `screenshot-1.png`, `screenshot-2.png`, `screenshot-3.png` - Optional screenshots for the frame gallery

## Background Color
The splash background color is set to `#000000` (black) in `config.build.js`.

## Required Images
Please add the required images to this directory before deploying as a Farcaster mini app.
